# create-hostic

Create a static web site using [Hostic](https://github.com/holtwick/hostic).

```bash
$ npm init hostic <project-name>
$ cd <project-name>
$ npm install
$ npm run dev
```

If using Yarn:

```bash
$ yarn create hostic <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
